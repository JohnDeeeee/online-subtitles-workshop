{TemplatePath}		add this before file path for example {TemplatePath}/img.jpg or {TemplatePath}/script.js

{SiteName}			add this between title tag for example <title>{SiteName}</title>

{SiteDescription}	add this between description tag for example <description>{SiteDescription}</description>

{SystemHeader}		this is very important add any were inside the head tag

{SystemFooter}		required by system keep before </body> 

{content}			add this where the pages content will appear
